js实现常用的排序算法

发表于 2018-03-05 | 分类于 JavaScript | | 阅读次数

面试的时候经常会让实现一些常用算法，这里我会将原理和代码都详细一一说明。

冒泡排序

原理: 冒泡排序原理是从依次将每一个数和后面的每一个数做一个对比，然后进行递减或者递增排序。
平均时间复杂度：O(n2)
空间复杂度：O(1)
稳定性：稳定

选择排序

原理：在要排序的一组数中，选出最小的一个数与第一个位置的数交换；然后在剩下的数当中再找最小的与第二个位置的数交换，如此循环到倒数第二个数和最后一个数比较为止。
平均时间复杂度：O(n2)
空间复杂度：O(1)
稳定性：不稳定

插入排序

原理：从数组第二个值开始，依次将后续的数值经过比较与前面排序后的序列比较后插入。
平均时间复杂度：O(n2)
空间复杂度：O(1)
稳定性：稳定

快速排序

原理：从数组中取一个值为基准值，并将剩下的值与之比较，小于基准值的放到左边，大于基准值的放到右边，并再次对左右两边进行快速排序，直至左右两边只剩一个元素
平均时间复杂度：O(nlog2n)
空间复杂度：O(nlog2n)
稳定性：不稳定
